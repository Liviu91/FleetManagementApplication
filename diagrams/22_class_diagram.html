Architecture Summary:
  • Frontend (Mobile): Xamarin/MAUI app with Auth, Route management, GPS logging, and OBD data collection via Bluetooth
  • Backend (ASP.NET Core): Controllers handling auth, admin, and driver operations
  • Data Layer: Entity Framework Core with Repository pattern
  • Messaging: RabbitMQ for async communication between mobile app and worker service
  • Background Processing: Worker service consuming OBD data from RabbitMQ and storing to database
